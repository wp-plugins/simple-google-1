== Description ==
insert Google +1 button to the ent of you posts.

== Installation ==

1. Download the file 'simple_google_button.zip'
2. Activate the plugin through the 'Plugins' menu in WordPress

that's it...

Manually:

1. Upload `j14-updates` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
